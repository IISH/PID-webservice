Class LocationType

java.lang.Object
  org.socialhistoryservices.pid.schema.LocationType

public class LocationType
extends java.lang.Object

locationType : a resolve Url. Just like the pidType.resolveUrl it points to a web resource. However the locationType is activated by extending parameters such as a -view to the handle request. Example: http://hdl.handle.net/12345/abcdefg?view=master For more information about the locationType, visit http://www.handle.net/ and read the section on the 10622/loc type.
